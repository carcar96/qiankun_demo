# Qiankun Demo
qiankun-base(vue2)  
qiankun-vue(vue2)  
qiankun-react(react18)  

## qiankun-base step:
cd qiankun-base  
npm install  
npm run serve  

## qiankun-vue step:
cd qiankun-vue  
npm install  
npm run serve  

## qiankun-react step:
cd qiankun-react  
npm install  
npm start  

## PS:
2个子服务和基服务都要分开启动  
qiankun-react 默认端口3000，可重新配置
